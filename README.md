🌦 Weather App (JavaScript + API):-

A simple and clean weather application that fetches real-time weather data using an API. Users can enter a city name and get current weather details like temperature, humidity, wind speed, and conditions.

live link :-


🚀 Features:-

🔍 Search weather by city name
🌡 Displays temperature in Celsius
💧 Shows humidity
🌬 Shows wind speed
🌥 Displays weather condition
⚠️ Error handling for invalid city input


🛠 Tech Stack:-

HTML
CSS
JavaScript (Async/Await)
Fetch API

📂 Project Structure:-

Weather-App/
│
├── index.html

⚙️ How It Works:-

User enters a city name
Clicks on the Submit button
App sends a request to the weather API
Displays weather data on the screen
Shows error if city is not found
